LISEO - Dynamic pricelist with visitors IP-Geolocation

Main features

  • This module geolocates a new customer and set the right pricelist according to its geolocation.
  • Geolocation is done via ipapi.co API and JSON returned is parsed by the module.
  • IP is cached during a configurable period to avoid asking geolocation for the same IP at each website page.
  • Activation of pricelist geolocation and configuration of cache IP duration is possible via the Website settings.

Support

Activate Pricelist based on customer's geolocation

  • In Website settings, under Pricelist configuration, activate Pricelist based on customer's geolocation
  • Activate Pricelists first if it is still disabled

Set IP caching duration

  • In Website settings, in "Shop - IP Cache" section, set the number of days for IP caching (1 day by default)

Pricelist configuration

  • In eCommerce > Products > Pricelists menu, you will access to Pricelists treeview
  • Select or create a Pricelist. In Pricelist form, under Configuration tab, select one or more Country Groups.
  • Countries are automatically computed in Countries field.
  • Go to Price Rules tab and configure prices for this Pricelist

Pricelist is loaded automatically at first visit of a customer

  • Customer sees prices based on its location according to pricelist configuration
